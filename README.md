Objectif du projet :
Concevoir et développer une application web permettant d'enregistrer, de gérer, de modifier, de rechercher et de supprimer les informations des étudiants. Le système vise à améliorer l'organisation des données, réduire les erreurs, gagner du temps et faciliter l'accès aux informations des étudiants par les utilisateurs autorisés.
Ce projet est réaliser par Saint-Aude, Ralph, Felix Nickenson, Dumace Marckenley et Konsilus Kerventz.
